UWEC Kaltura V2 → V7 Migration Tool

Instructions

This tool will convert Kaltura V2 iframe code to V7 iframe code. This tool will only work for videos that were embedded using the rich content editor Kaltura tool. If you used embed links directly from Kaltura/My Media or tiny URLs, this tool probably won't work for you.

To get going, you'll need the old Kaltura iframe code and the new dimensions you want to use. To learn how to do this, expand the instructions below. They're a bit lengthy, but once you get the hang of it, this process will be quick and easy.

Getting the old iframe code

To get the iframe code you want to convert, access the page/discussion/assignment etc. in Canvas that has the embedded video. In the lower right corner of the text area containing the video, click the button that looks like </>. This will switch the editor to the HTML view mode. This might look scary, but you can always get back by clicking the same button.

From within the HTML viewer, look for blocks of code that look like this:

This will be long, possibly occupying multiple lines. Copy the entire block of code. If you start copying at the beginning of the <iframe> tag and end at the end of the </iframe> tag, you should be good to go. If you include the <p> tags, that's fine too; just make sure you get both the opening and closing tags and nothing beyond them.

Getting the updated iframe code

With the old iframe code copied, navigate back to this page and paste it into the "Old Kaltura iframe" text area below. It should automatically populate the "Updated Kaltura iframe code" text area with the new code. You can also update the dimensions of the video by selecting the desired dimensions from the dropdown menu, and the new code will automatically update. Which dimensions are "correct" depends on your use case. If you're not sure, ask your friendly neighborhood instructional designer.

Once the new code is generated, click the "Copy" button to copy the new code to your clipboard. It will be long, so don't bother trying to highlight and copy; just trust the button.

Updating the video player in Canvas

With the new code copied, go back to the Canvas page editor and paste the new code in the same place you found the old code. If you are just getting started, consider pasting this below the old code rather than replacing it to make sure everything works as expected. Once you're satisfied, you can remove the old code (or delete the video in the rich content editor). Remember, to get back to the rich text editor, click the </> button again; your video should now be visible in the editor using the new player, possibly along with the old player if you left both pieces of code.

If you have multiple videos to convert, you should fix them one at a time or all at once using this process.
